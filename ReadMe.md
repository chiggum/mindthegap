**How to use**

For Linux
2 simple steps:
- chmod +x path/to/install.sh
- /path/to/install.sh

That's it! Now just write in terminal: mindthegap -h
The above command will show you how to use the software.

For Windows
- open cmd
- go to the directory contatining mindthegap.exe
- type: mindthegap.exe -h
- The above command will show you how to use the software.
